Friday March 19, 2021

Developer baru untuk pengembangan Star Citizen

by Albert Gunawan

Cloud Imperium mengumumkan "FIRESPRITE" sebagai developer tambahan dalam tahap pengembangan game ini. Firesprite sendiri merupakan game studio yang berbasis di inggirs yang telah menghasilkan "The Playroom" serta game horror dengan judul "The Presistence".

Cloud Imperium sendiri memiliki banyak studio yang tersebar di Los Angeles, Austin, Frankfurt, Wilmslow, dan Derby, dan telah bekerja dengan sejumlah kontraktor selama satu dekade dalam pengembangan Star Citizen, termasuk studio Dead by Daylight oleh Behavior Interactive dan perusahaan produksi Virtuos.
